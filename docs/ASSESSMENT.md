# Divergent Technologies Ltd. — Frontend Developer Technical Assessment
## Restaurant POS – New Order Module

### Objective
Evaluate frontend development skills including Next.js, React, TypeScript, API integration, UI implementation, state management, code organization and overall development practices.

### Technology Stack
- Next.js (App Router) / React
- TypeScript (Preferred) / JavaScript
- Tailwind CSS
- shadcn/ui

### API
Base URL: `https://dummyjson.com`

- `GET /products`
- `GET /products/search?q={query}`
- `GET /products/categories`
- `GET /products/category/{category}`
- `GET /products/{id}`
- `POST /carts/add`

### Functional Requirements
1. Display product grid with image, name, price, rating, stock, category and Add to Cart.
2. Debounced search (300ms).
3. Category filter.
4. Product details page.
5. Shopping cart with add/remove/update quantity and localStorage persistence.
6. Order summary (Subtotal, VAT, Discount, Coupon, Grand Total).
7. Coupon support (SAVE10, SAVE20, WELCOME).
8. Checkout confirmation modal.
9. POST order to API.
10. Loading, empty and error states.
11. Responsive design.

### Bonus Features (Optional)
- Infinite Scroll
- Dark Mode
- Undo Remove
- Optimistic UI
- Virtualized List
- Wishlist
- Keyboard Shortcuts
- Unit Tests
